python条件语句

条件语句

if 语句
if 语句用于条件控制
if 语句包含零个或多个 elif 子句及可选的 else 子句
关键字 elif 是 else if 的缩写；Python 中只能使用 elif
if 语句可以嵌套
代码示例：

python

score = int(input(‘请输入分数：’))

if 0 <= score <= 100:
if score < 60:
print(‘不及格’)
elif score < 80:
print(‘良’)
else:
print(‘优’)
else:
print(‘分数输入错误，分数范围为：0~100’)
